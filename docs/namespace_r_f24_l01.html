◆ getStatus()

uint8_t RF24L01::getStatus ( )

get the RF24L01 STATUS register

Returns
STATUS register 0-7 bit [TX_FULL,RX_P_NO,RX_P_NO,RX_P_NO,MAX_RT,TX_DS,RX_DR,Reserved] TX_FULL is set when the tx buffer is full. RX_P_NO is the Rx pipe number on which the next call to getReceiveMsg will return data from (handy for multi per communication). MAX_RT indicates a transmission timeout this flag is cleared by transmitMsg for you, said call will return false instead. TX_DS indicates that the transmission was success full, again transmitMsg will handle it for you. RX_DR indicates that there is Rx FIFO data ready to be read.
